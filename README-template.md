### What I learned

- how to turn an <a> to a button through css

```html
<a href="https://google.com" class="button">Go to Google</a>
```

```css
a.button {
  -webkit-appearance: button;
  -moz-appearance: button;
  appearance: button;

  text-decoration: none;
  color: initial;
}
```
